@core/meta/タスク分解と保存ルール.md

以下を実行してください：

1. `git status` で変更内容を確認
2. ユーザーに「保存する内容のサマリ」を日本語で表示
3. inputs/ 配下が含まれていないことを確認（含まれていたら警告して中断）
4. `git add` で対象ファイルをステージ（inputs/ は除外）
5. コミットメッセージは Claude が日本語で生成（例: "業務棚卸し: 経費精算 - 工程を記録"）
6. `git commit` を実行
7. 結果を一行で報告
