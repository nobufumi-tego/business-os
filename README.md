# 業務改善OS スターター

日本の DX担当者向け、業務改善のための Claude Code スターター。
Windows 11 / PowerShell 未経験から、5分で最初の棚卸しを始められます。

## 始める前に確認すること

| 項目 | 条件 |
|------|------|
| OS | Windows 11（Windows 10 は動作未確認） |
| 権限 | 管理者アカウント、または PowerShell を管理者として実行できること |
| Claude プラン | Claude Pro 以上のサブスクリプション、または Anthropic API キー |
| 費用 | サブスクリプションは月額固定、API キーは利用量に応じた従量課金 |

> 認証は基本的に `claude` 初回起動時に開くブラウザでサインインします（推奨）。
> API キー方式を使う場合の手順は [docs/インストール.md](docs/インストール.md) を参照。

## まず動かしてみる（5分）

### 1. 必要なものをインストール（初回だけ）

PowerShell を「管理者として実行」で開いて、以下をコピペ：

```powershell
winget install OpenJS.NodeJS.LTS Git.Git astral-sh.uv
npm install -g @anthropic-ai/claude-code
```

### 2. このスターターを取得

```powershell
git clone https://github.com/nobufumi-tego/business-os.git
cd business-os
.\setup.ps1
```

### 3. Claude を起動して、まず一言だけ話しかける

```powershell
claude
```

**初回はブラウザが開いて Claude のサインインを求められます。** サインインが完了すると、文字を打ち込める入力欄が表示されます。

ここからが本題です。Claude Code は、皆さんが**何か話しかけるまで黙って待つ**しくみです。
練習もかねて、まず **「こんにちは。」** と最後の句点まで打って Enter を押してみてください。
（「はじめる。」「？」「助けて。」など、何でも構いません。Enter を押した時点で送信されるので、**文として終わってから** Enter を押すのがコツです）

Claude が使い方を一言説明して、続けてメニューを出してくれます。
番号で選んで3問答えれば、最初の棚卸しシートが完成します。

> うまく動かない場合は [docs/インストール.md](docs/インストール.md)（詳しい版）と [docs/困ったとき.md](docs/困ったとき.md) を確認してください。

## 何ができるか

| コマンド | 用途 |
|---|---|
| `/業務棚卸し` | 業務の中身を対話で構造化する |
| `/ドキュメント棚卸し` | フォルダの中のファイルを整理する |
| `/システム棚卸し` | 既存システムを整理して改造案を作る |
| `/カスタマイズ` | 自社用の専用コマンドを Claude と一緒に作る |
| `/保存` `/履歴` `/巻き戻し` | Git をシンプルに使う |
