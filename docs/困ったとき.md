# 困ったとき

## まずこれを試してください

### エラーが出たら、Claude に貼り付けて聞く

```
[エラーメッセージをそのままコピー]
これ何？どうしたら直る？
```

これで9割解決します。

## それでもダメな場合

### 環境が壊れている疑いがあるとき

PowerShell でこれを実行：

```powershell
.\doctor.ps1
```

足りないツール・設定が一覧で出ます。それを Claude に渡して聞いてください。

### Claude が起動しない

- ブラウザのサインインが完了しているか
- Pro プランの quota 切れではないか（Anthropic の管理画面で確認）

### 文字化けする

PowerShell のコードページが Shift-JIS のままの可能性。

```powershell
chcp 65001
```

を実行してから再試行。

### inputs/ のファイルが間違って commit されそう

`/保存` 前に Claude が警告を出します。
万一 commit してしまったら、`/巻き戻し` で1つ前に戻せます。

### 文章を途中で改行したい / 入力途中で Enter を押してしまった

- **複数行を一度に書きたい場合**：`Shift + Enter` で改行できます（Enter だけだと送信されてしまうため）
- **文の途中で誤って Enter を押してしまった場合**：そのまま続きを次の行に書けば、Claude が前後をつなげて読み取ってくれます。慌てず続きを書き、最後に Enter を押せば大丈夫です
- **明らかに意図と違う回答が返ってきたら**：「すみません、さっきのは入力途中でした。正しくは ◯◯ です」と書き直せば、Claude が訂正します

## 最後の手段

すべてをやり直したい場合、フォルダごと削除して再 clone してください。
ただし `outputs/` は失われるので、必要なら先にバックアップを取ってください。
