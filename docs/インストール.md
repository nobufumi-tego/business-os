# インストール手順（詳しい版）

## 必要なもの
- Windows 11
- インターネット接続
- 管理者権限（PowerShell を管理者として実行できること）

## 手順

### 1. PowerShell を管理者として実行で開く

スタートメニューから「PowerShell」を探し、右クリック→「管理者として実行」。

### 2. 実行ポリシーを変更（初回だけ）

```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

「Y」を押して確定。

### 3. 必要なツールをインストール

```powershell
winget install OpenJS.NodeJS.LTS Git.Git astral-sh.uv
```

3つすべて入るまで待ってください。

### 4. Claude Code をインストール

```powershell
npm install -g @anthropic-ai/claude-code
```

### 5. このスターターをダウンロード

```powershell
cd $HOME\Documents
git clone https://github.com/nobufumi-tego/business-os.git
cd business-os
```

### 6. セットアップを実行

```powershell
.\setup.ps1
```

途中で名前とメールアドレスを聞かれます。Git の記録に使われるだけなので、社内で使っているもので OK。

### 7. Claude を起動

```powershell
claude
```

初回はブラウザが開いて Claude のサインインを求められます。
社内 Wi-Fi で繋がらない場合はスマホテザリングを試してください。

サインインが完了して、文字を打ち込める入力欄が出たら成功です。
最初に **「こんにちは。」** と最後の句点まで打って Enter を押してみてください。Claude が使い方とメニューを案内します（何を打っても構いません。練習だと思って。Enter を押した瞬間に送信されるので、**文として書ききってから** Enter）。
