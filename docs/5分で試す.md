# 5分で試す（チュートリアル）

> **使い方のコツ**：Enter を押すと Claude にその時点の文章が**送信**されます。
> 文章は1行で書ききってから Enter を押してください（途中で押してしまっても、続きを書けば Claude が察してくれます）。

セットアップが終わったら、まず「業務棚卸し」を試してみましょう。

## 1. Claude を起動して、まず挨拶する

```powershell
claude
```

起動しただけでは Claude は何も言いません。**こちらから先に話しかける**しくみです。
練習として、まず **「こんにちは。」** と最後の句点まで打って Enter を押してください。
（「はじめる。」「？」など何でもOK。短くても、文として終わっていれば大丈夫です）

Claude が使い方を一言説明して、メニューを出してくれます。

## 2. メニューから「1」を選ぶ

```
何をしますか？番号で答えてください。
  1) 業務棚卸し
  ...
```

`1` と入力して Enter。

## 3. 3問だけ答える

Claude から以下のような質問が来ます。テキトーで構いません。

- Q: どんな業務を棚卸ししますか？
  A: 例「経費精算」
- Q: その業務の目的は？
  A: 例「月末の経費を漏れなく処理する」
- Q: 担当部署は？
  A: 例「経理部」

## 4. ファイルが出来ているか確認

エクスプローラで以下を開いてみてください：

```
business-os\outputs\業務棚卸し\
```

`2026-04-29_経費精算.yaml` のようなファイルが出来ているはずです。
これで最初の棚卸しシートが完成。

## 5. 続きをやるか保存するか選ぶ

Claude から「続けて深掘りしますか？それとも一旦保存しますか？」と聞かれます。

- 続ける → 工程・関係者・困りごとを聞いていく
- 保存する → `/保存` でローカル Git に記録される

## 次のステップ

- 同じ手順を **本物の業務**で試してみる
- ドキュメントの整理もしたければ `/ドキュメント棚卸し`
- 自社専用のコマンドを作りたければ `/カスタマイズ`
